# IPListener
IPListener is a program for capturing, viewing, and analyzing network packets. You can build it yourself or use it in your own projects.

It uses pcap version 2.4.0, the latest version.

This program captures entire network packets, allowing you to analyze them later, for example, using an AI agent.

Feel free to follow us on Telegram, where you can find the latest compiled versions: https://t.me/KernelaStudio. We'd love to hear from you!

After running the program as administrator (this is the only way to intercept), select Device and enter the device ID.

After this, if traffic passes through this device, sniffing will begin. The packet will be written to packets.txt in the same repository as the .exe executable.
